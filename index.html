系统说明与协同作战指南
① 织带系统
智能多空织带（EMA 12/26/50）— 前线侦察兵
EMA12 > EMA26 > EMA50，价格突破织带。最灵敏，适合20-30%轻仓试错。
② 维加斯隧道
维加斯机构隧道（EMA 144/169/250）— 总司令部
价格 > EMA144 > EMA169，EMA250向上。一旦触发往往代表大级别趋势。
③ 顾比均线
顾比均线 GMMA（多均线群）— 多兵种协同
6条短期EMA群全面穿越6条长期EMA群。右侧共振确认，加仓至50%。
④ Alpha-Squeeze
Alpha-Squeeze（VWMA + 布林带挤压）— 量子雷达
布林带收缩进肯特纳通道后释放，VWMA向上。无量空涨被过滤，Squeeze释放=主升浪信号。
🔥 四维共振 = 神级买点
四个系统同时发出买入信号，多维共振。往往是主升浪起点。
协同仓位：① 织带→20-30%；② 顾比→50%；③ Squeeze释放→重仓；④ 维加斯确认→长线定海神针。
🎯 触发强度评分系统（0-100分）
每只股票显示 0-100 综合质量分，区分「刚刚突破」和「放量强势突破」的质量差异。结果页排序可选「质量分↓」。
📈
① 突破幅度 （0-20分）
价格突破关键均线（EMA12/26/50）的幅度。突破 0.5%=7分 · 1%=13分 · 2%=18分
⚠ 超过5%反而扣分（追高风险）
满分 20
📊
② 成交量倍数 （0-25分）
今日成交量 ÷ 20日均量。1.5倍=10分 · 2倍=17分 · 3倍+=25分
机构放量突破是最强信号，权重最高
满分 25
⚡
③ 量化动能 （0-20分）
Alpha-Squeeze 归一化动能值（SMOM÷ATR）。动能>0且加速=10-20分
配合 Squeeze 释放时动能最纯净
满分 20
🔥
④ 共振数量 （0-20分）
四套指标同时发出买入信号的数量
1/4=5分 · 2/4=10分 · 3/4=16分 · 4/4=20分
满分 20
📍
⑤ 价格位置 （0-15分）
价格距离维加斯隧道 / EMA250 的距离
刚站上支撑=15分（风险回报最优） · 远离支撑=2-5分
满分 15
综合评级 80-100 · A+ 极强 65-79 · A 强 45-64 · B 中等 25-44 · C 弱 0-24 · D 不推荐
💡 实战建议：优先关注 质量分 ≥60 + 共振 ≥3/4 的股票，这类信号胜率和赔率最优。质量分只对有买入信号的股票计算。

🛡 风控工作台使用指南
扫描完成后勾选股票 → 点「🛡 批量送入风控」→ 填入账户参数 → 系统自动计算最优仓位和止损价。算完之后可以点「➕ 加入自选股」把勾选的股票一键加入⭐自选股（写入日期），方便后续跟踪。
📌 止损模式说明
摆动低点止损 — 最近5根K线最低价×0.995，最贴近市场结构，适合波段交易
维加斯隧道下轨 — EMA169×0.99，趋势交易首选，止损空间较大
ATR×1.5止损 ★推荐 — 14日平均波幅×1.5，数学最优，适合大多数股票
ATR×2.0止损 — 止损更宽松，给股票更多波动空间
固定3%止损 — 简单直接，适合低波动蓝筹股
固定5%止损 — 适合高波动成长股
📌 其他参数说明
止盈目标 1.5R/2R/3R — 风险回报比。2R=赚的是亏的2倍，长期胜率只需>33%就能盈利
单笔最大风险% — 建议1-2%，每笔亏损不超过账户的1-2%
单股资金上限% — 建议≤25%，防止单一股票过度集中
行业集中度上限% — 同一板块总仓位上限，防止行业集中风险，建议≤30%
ATR周期 — 默认14日，数字越小越敏感，越大越平滑
💡 黄金公式：建议股数 = (账户×单笔风险%) ÷ 每股风险 · 所有状态显示「✓ 正常」才是安全仓位

📊 ETF池
核心ETF / 行业ETF / 因子ETF / 国际ETF，各自可单独关闭
🎯 综合大池

👤 我的股票池
你的⭐自选股 + 📦持仓股（自动加入，不用手动维护）
💡 想缩小扫描范围？去扫描参数页把不需要的池子点掉即可，输入框里的股票代码列表会跟着实时更新。
⭐📦 自选股 / 持仓股 使用指南
⭐ 自选股（观察池）
加股票时可手动选加入日期（默认今天，可补录之前的日期）
筛选栏按共振维度分类，适合从候选池里挑买入机会
买卖信号：🟢买入信号 / ⚠信号减弱 / 🔴卖出 / 观望
点⚡可自动算止损/目标价（ATR模型）
📦 持仓股（真实仓位）
筛选栏按仓位状态分类：卖出预警/信号减弱/浮亏/浮盈
💰卖出＝真实交易，支持部分卖出，自动结算已实现盈亏；✕删除＝纯粹从列表移除（不代表真实成交，不结算盈亏）
数据90秒内有缓存，重复切页不会重新拉取；点「📡扫描全部」强制刷新
点股票代码下方备注区可编辑备注/标签
📲 Telegram / Discord 推送说明
在扫描参数页配置好 Bot Token / Webhook 并勾选启用后，每次扫描完成会自动推送，无需手动操作。推送内容跟结果页排序逻辑一致（共振→质量分→量能值）。
Telegram设置
① @BotFather 创建Bot拿Token ② 先给Bot发一条消息 ③ 填Token后点「🔍自动获取」拿Chat ID
Discord设置
频道设置 → 整合 → Webhook → 创建Webhook → 复制URL粘贴进去
推送范围
所有有信号(≥1/4) / 三维以上共振(≥3/4) / 仅四维共振(4/4) / 质量分≥60，四选一
自动驾驶模式
设定时间自动扫描+推送，无需手动点击，可关屏后台运行；推荐勾 15:45(黄金窗口，数据基本锁定且留反应时间) + 16:05(收盘后100%确认，适合定次日计划)，两个搭配=日内快反应+次日定计划
风控执行单
风控页勾选股票 → 「📋今日执行单」查看 → 可再单独推送到 Telegram/Discord
跨设备同步
自选股页「⬇导出/⬆导入」可把自选股+持仓数据导出成JSON，换设备后导入
